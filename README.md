# 🇮🇳 Bharat Sahayak AI

Bharat Sahayak AI is an intelligent Android AI assistant application developed using modern technologies. The project aims to provide smart assistance, voice interaction, and productivity features for users.

## 🚀 Features

- 🤖 AI Chat Assistant
- 🎙️ Voice Command Support
- 🧠 Smart Response System
- 📱 Modern Android UI
- ⚡ Fast Performance
- 🔒 Secure and User-Friendly

## 🛠️ Tech Stack

- Kotlin
- Android Studio
- Jetpack Compose
- Android SDK
- Git & GitHub
